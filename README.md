# MokaScrapper
- 구직 사이트 stackoverflow에서 원하는 검색 결과를 스크랩하여 화면에 띄우고 csv파일로 저장하는 사이트 입니다 

## 기능
1. 검색 결과로 이름, 회사정보, 링크 등 각종 정보 스크랩 후 페이지에 출력
2. 검색한 결과를 csv파일로 저장
3. 스크랩 결과를 저장하여 다음 검색 시 빠르게 결과 출력
